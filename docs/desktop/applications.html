Default Applications

Oreon comes with a carefully selected suite of applications designed to provide a complete desktop experience out of the box.

Web Browser

Mozilla Firefox

A popular, open-source browser known for its speed, security features, privacy focus, and extensive customization options through add-ons.

File Manager

Nautilus (Files)

Intuitive graphical interface for browsing folders and files on your computer, external drives, and network locations.

Text Editor

Gedit

Lightweight yet capable editor suitable for taking notes, editing configuration files, and writing basic scripts.

Terminal

GNOME Terminal

Access to the powerful Linux command line with support for multiple tabs, customizable profiles, and advanced features.

System Settings

GNOME Control Center

Centralized location to manage network connections, display settings, sound devices, user accounts, and more.

Image Viewer

Eye of GNOME

Simple and elegant image viewer supporting various image formats with basic editing capabilities.

Key Features

Firefox Browser

  • Advanced privacy and security features
  • Extensive add-on ecosystem
  • Bookmark and password synchronization
  • Developer tools integration

File Manager

  • Multiple view options (icons, list)
  • Tabbed browsing
  • Built-in file search
  • Network location support

Text Editor

  • Syntax highlighting for multiple languages
  • Plugin support for extended functionality
  • Search and replace capabilities
  • Automatic saving

Terminal

  • Multiple tabs and windows
  • Customizable color schemes
  • Unlimited scrollback buffer
  • Profile management
