Software Center

Oreon provides a graphical Software Center, making it easy to discover, install, and manage applications without needing to use the command line.

Easy Software Management

The Software Center serves as a user-friendly storefront for browsing, installing, updating, and removing applications with just a few clicks.

Getting Started

The Oreon Software Center provides a user-friendly graphical interface for managing software on your system. Under the hood, it utilizes the system's package manager (dnf) and integrates with other packaging formats like Flatpak.

Traditional Packages

RPM packages managed by DNF, providing deep system integration and performance.

Flatpak Apps

Sandboxed applications from Flathub, offering enhanced security and broader software selection.

Browsing and Searching

Explore Categories

Browse featured applications and categories like Audio & Video, Graphics, Office, Development Tools, and Games.

Search Applications

Use the search bar to find applications by name or keywords. Results update dynamically as you type.

View Details

Click on any application to see descriptions, screenshots, version information, and user reviews.

Installing Applications

Installing software through the Software Center is straightforward:

Installation Process

  1. Select an application from browse or search
  2. Review the application details page
  3. Click the "Install" button
  4. Enter your password when prompted
  5. Wait for download and installation to complete

What Happens

  • Dependencies are automatically resolved
  • Progress indicator shows download status
  • Application appears in your menu when complete
  • Desktop shortcuts may be created

Managing Installed Applications

The "Installed" section lists all applications currently on your system that are managed by the package manager or Flatpak. From here you can:

  • View installed application details
  • Remove unwanted applications
  • Check application versions
  • Review installation dates

To remove an application:

  1. Navigate to the "Installed" section
  2. Find the application you want to remove
  3. Click "Remove" or "Uninstall"
  4. Confirm the action with your password
  5. The application and its data will be removed

Updating Software

Keeping your applications up-to-date is crucial for security and functionality.

Update Process

Automatic Checks

  • • Software Center checks for updates periodically
  • • Notifications appear when updates are available
  • • Both system and application updates are detected

Manual Updates

  • • Visit the "Updates" section
  • • Review available updates
  • • Click "Update All" or select individual updates
  • • Authenticate with your password

Software Sources

The applications available in the Software Center come from configured software repositories and packaging systems.

Oreon Repositories

Official packages tested and optimized for Oreon Linux

EPEL Repository

Extra packages for Enterprise Linux compatibility

Flathub

Sandboxed applications with enhanced security

Pro Tip

Flatpak applications run in a sandboxed environment for enhanced security, while traditional RPM packages provide deeper system integration. Choose the format that best suits your needs!
