Frequently Asked Questions

Common questions and answers about Oreon Linux

What are the system requirements for Oreon Linux?

Oreon Linux requires a 64-bit processor, 2GB RAM (4GB recommended), 20GB disk space, and a graphics card compatible with GNOME.

How long is each Oreon version supported?

Oreon follows the support timeline of its base RHEL version, with security updates provided throughout the entire support period. See each version's support timeline for specific dates.

Can I upgrade from one Oreon version to another?

We are working on this, but it is not yet available.

Is Oreon Linux free to use?

Yes, Oreon Linux is free to use for both personal and commercial purposes, in accordance with the licenses of included software.

How can I contribute to the Oreon Project?

You can contribute by testing, reporting bugs, helping with documentation, or developing. Join our Discord community to learn more about contribution opportunities.
