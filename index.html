Enterprise Linux but a little friendlier

Our primary goal is to make Enterprise Linux more suitable towards use on desktops and laptops. We focus on providing the best user experience right out of the gate.

Friendly Defaults

Oreon ships with the GNOME desktop by default with a few tweaks and extra applications to fulfill the user's needs.

Continuous Updates

With the help of upstream and our team, we are able to promise several years of security fixes and new features to each Oreon release.

Built on Enterprise Linux

Oreon is based on RHEL and compatible with all its packages. Take advantage of the stability and security of Enterprise Linux.

Current Versions

Oreon Linux is available in multiple versions to suit your needs.
