What This Means

Mainstream Support Phase (2024-2027)

  • Regular feature updates and improvements
  • Security patches and bug fixes
  • Hardware enablement updates
  • Full community and commercial support

Extended Support Phase (2027-2032)

  • Critical security updates only
  • No new features or hardware enablement
  • Community support continues
  • Migration path to newer versions recommended

End of Life (After May 2032)

  • No further updates or patches
  • Migration to newer Oreon version required
  • Community may provide unofficial support
